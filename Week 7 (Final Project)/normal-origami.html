Normal Origami

Here, you’ll find your average, every day origami. If you’re just starting out in the world of origami, or just looking for something easy to do, check out some of these projects. Most if not all of these projects have comprehensive tutorials that will help you fold to your heart's desire. With the skills you learn from these projects, you'll be able to manage pretty much anything. Of course, once you have enough practice with this level of origami, you should be ready to poke around in some of the 3D origami.

Simple Turtles

These turtles are an adorable little project you can do in 5-10 minutes. They can be any size you want, and are fairly easy to make. Definitely beginner friendly, and good for when you don’t have a lot of time on your hands.

Tutorial: https://www.youtube.com/watch?v=LbUI7lqO5t0

Sitting Fairy

This fairy is a bit harder of a project. It requires some precise folds and a bit of patience, so take this project on if you’re looking for a bit of a challenge. This project takes about 45 minutes to an hour, so make sure you have time to finish it before you begin.

Tutorial: https://www.youtube.com/watch?v=ySSTMeOQLKc

String of Cranes

This project is pretty easy, but it might take more time than some of the others. This string of cranes is part of something called a Senbazuru, which means "1000 cranes" in Japanese. There is a Japanese legend that says if you fold 1000 paper cranes, then the gods will grant you a wish. I wanted to do this, and this is part of the Senbazuru that I made. All you have to do is learn how to fold an origami crane, and then do it over and over again.

Tutorial: https://www.youtube.com/watch?v=KfnyopxdJXQ
